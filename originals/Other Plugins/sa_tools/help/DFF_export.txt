 этот скрипт предназначен только для экспорта ОБЪЕКТОВ КАРТЫ. Не работает с тачками и педами.....

Дневной свет на модели надо записывать в канал vertex color, а ночной - в канал
vertex illumination...
Когда пользуетесь модификатором vertex paint , то там в секции PARAMETERS \ CHANNEL
есть  переключатель этих каналов.

Ночной свет имеет 4 параметра, по дефалу стоят средние...

Есть еще переключатель экспорта ("type 1", "type 2").. Так вот лучше пользуйтесь "type 1" -
он правильней работает...

если включён флажок multi object export , то можно экспортнуть сразу много объектов...
экспорт делается в указаную папку по именам выбраных объектов... Например есть объекты
"zemla01", "zemla02", ....., "zemla99", "DOM01", ...., "DOM09"... То и файлы будут
называться соответственно...

Есть еще флажок geometry light - еси он выключен, то будет эффект как в тёмных туннелях СА
Для обычных объектов флажок включен...

На панели скриптов есть секция "show vertex color" и две кнопки "День" "Ночь"... Этими
кнопками можно менять канал отображаемый на экране для выбраных объектов... Другими словами
если уже сделан прелит и день, и ночь... то можно легко посмотреть разницу.

совет: для лучшей видимости прелита, во время его создания отключите текстуры. Делается это так -
       во вкладке DISPLAY в группе SHADED надо выбрать пимпочку Object Color...

This script only for map object export (dont work with peds and cars)

-- You can export VC and NVC from one model

!!! DAY PRELIT you should put to VertexColor channel
!!! NIGHT PRELIT you should put to VertexIllumination channel
(if you use VertexPaint modificator, choose needed channel in parameters section)

-- supports multiexport. If its ON then files wil be names same as exported objects in
   directory you choose

-- 2 types for export... use just "type1". 2nd is worse

-- If you turn off GEOMETRY LIGHTS flag - you can see effect, like SA tunnels (very dark)
